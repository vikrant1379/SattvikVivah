Design prompt: Profile card typography spec

Overall font stack:

font-family: Inter, system-ui, -apple-system, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif.

Use optical sizing where available; prefer variable fonts if supported.

font-weight: 500

line-height: 16px

letter-spacing: 0

color: #667085

spacing: 4px margin-bottom

font-weight: 700

letter-spacing: 0

color: #101828

spacing: 6px margin-bottom

spacing: 8px margin-bottom

Accessibility and truncation

Minimum contrast: WCAG AA (≥4.5:1 for body, ≥3:1 for bold).

Truncate long names and locations with ellipsis after 1 line; provide title attributes for hover.

Keep tap targets ≥44px height on mobile.

Spacing and layout rhythm

Vertical rhythm: 4px base unit.

Section paddings: 12–16px internally; 16–20px card padding.

Keep 8–12px spacing around separators and between icon-text pairs.

Responsive adjustments

Mobile (≤375px width):

Name: 22–24px/30px

Body rows: 13–14px/18–20px

Reduce item gaps by ~2px

Tablet/desktop:

Name up to 30px

Keep body at 14–15px

Increase action gaps for clarity

Optional alternate font pairing

Display (name): 'Poppins' or 'Nunito Sans' at 700–800 weight

Body/UI: 'Inter' or 'Roboto'
